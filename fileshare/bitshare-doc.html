API Documentation

Bitshare: OrcaNet-Tuna's File Sharing API

Bitshare is a decentrailized p2p filesharing platform built on libp2p. This documentation contains information about how to make requests using our API. Use headers and the request body format included in examples.

POST PROVIDE_FILE
http://localhost:8081/rpc

Make a file available for download at set price

Request Body

  • Type: Raw

  • Description: JSON object with the following keys:

    • jsonrpc (string): The version of the JSON-RPC protocol. Do not change this.

    • method (string): The method to be invoked. Do not change this.

    • params (array): An array of parameters for the method.

      • file_path (string): The path of the file to be shared.

      • price (number): The price set for the file.

    • id (number): The request identifier.

HEADERS
Content
application/json

None

BODY raw
                                                  
                                                    
{
    "jsonrpc": "2.0",
    "method": "FileShareService.ProvideFile",
    "params": [
        {
            "file_path": "media/test2.txt",
            "price": 3
        }
    ],
    "id": 1
}

POST GET_UPDATES
http://localhost:8081/rpc

File Share Service Updates

This endpoint allows you to retrieve updates for the File Share Service. Call this at regular intervals.

Request Body (Don’t change)

  • jsonrpc (string, required): The JSON-RPC protocol version.

  • method (string, required): The method to be called, in this case, “FileShareService.GetUpdates”.

  • params (array, required): An array of parameters for the method.

  • id (number, required): An identifier established by the client.

Example:

``` json
{
“jsonrpc”: “2.0”,
“method”: “FileShareService.GetUpdates”,
“params”: [],
“id”: 1
}

```

Response

The response will contain the following fields:

  • result (object): The result of the method call.

    • success (boolean): Indicates if the operation was successful.

    • peer_id (string): The peer ID.

    • multi_addr (string): The multi address.

    • status (string): The status.

    • private_ip (boolean): Indicates if the IP is private.

    • providing (array): An array of objects containing file information.

      • FileName (string): The name of the file.

      • FileSize (number): The size of the file.

      • FileHash (string): The hash of the file.

      • FileExtension (string): The extension of the file.

      • Multiaddress (string): The multi address of the file.

      • PeerID (string): The peer ID of the file.

      • RequestID (string): The request ID of the file.

    • requested_files (array): An array of requested files.

    • downloads (array): An array of downloaded files.

  • error (null): Indicates if an error occurred.

  • id (number): The identifier established by the client.

HEADERS
Content
application/json

None

BODY raw
                                                  
                                                    
{
    "jsonrpc": "2.0",
    "method": "FileShareService.GetUpdates",
    "params": [],
    "id": 1
}
